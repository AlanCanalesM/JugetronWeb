Comandos para realizar el pull y el push!!!

Descargar el proyecto de inicio
1.- git config --global user.mail "aqui su correo"
2.- git config --global user.main "su nombre de usuario"
3.- git init
4.- git remote add origin https://github.com/AlanCanalesM/JugetronWeb.git
5.- git pull

Hacer push para los cambios
1.- git status
2.- git add .
3.- git commit -m "aqui ponen entre comillas la descripci√≥n de sus cambios"
4.- git branch -M main
